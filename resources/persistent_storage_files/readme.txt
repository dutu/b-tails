Q: I have started the computer from this memory stick/card and I need to send some bitcoin. How do I do it?
A: You use the "Electrum Bitcoin Wallet" application. Follow the instruction at https://dutu.github.io/b-tails/guide/use/electrum.html

Q: I need to locate the bitcoin wallet file. Where is it?
A: The bitcoin wallet file is on VeraCrypt encrypted partition. To open it in Electrum, follow the instruction above.
