
Q: How do I use this memory stick?
A: You need to insert it into a computer USB port and start up the computer from USB.

Q: Is there a detailed instruction on how to start up a computer from this USB memory stick/card?
A: Check https://dutu.github.io/b-tails/guide/use/on_tails.html

Q: During the process I need a password to "Unlock Persistent Storage". How do I find the password?
A: Check the file `passwords.txt`.

Q: I also need a password to "Unlock VeraCrypt Volume". How do I find the password?
A: Follow these steps: 1) Boot up the computer from this USB drive. 2) Choose "Places -> Persistent" 3) Check the file `passwords.txt`.

Q: Is there additional information stored on this memory stick/card?
A: Look on Persistent Storage and VeraCrypt encrypted partition.

A: Goto https://dutu.github.io/b-tails/
